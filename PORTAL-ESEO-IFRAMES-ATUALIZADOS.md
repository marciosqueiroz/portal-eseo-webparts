| `PREVIA-LOCAL.html` | Arquivo apenas para teste local |

## Padrão para futuras atualizações

- Manter a publicação na branch `main`.
- Substituir o arquivo existente, preservando exatamente o mesmo nome.
- Não criar nomes com `v4`, `v5`, `final`, `novo` ou datas.
- Não acrescentar uma segunda extensão `.html`.
- Não usar `sandbox` nos iframes, pois pode impedir links em nova guia.
- Manter `Enforce HTTPS` habilitado no GitHub Pages.
